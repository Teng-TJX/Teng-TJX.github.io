二维数组和多维数组

  • 一维数组的单元中，保存了一个数组，那么该数组就是二维数组
  • 二维数组可以想象成一个表格
  • 二维数组同样也用下标来进行控制
  • 二维数组常用来存储一组相似/相关的数据
  • 二维数组的每个单元都保存一个一维数组，就是三维数组；三维数组的每个单元都保存一个数组就是四维数组； 依次类推，可以到n维。在一般开发中，最常用的是一维和二维数组，三维极少使用。
