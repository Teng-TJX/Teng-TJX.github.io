05-PHP变量

  • PHP

变量

变量命名规则：

  • 由字母、数字、下划线组成，必须以字母或者下划线起头
  • 变量的长度可以是任意的，但不能超过255个字符
  • 最前面需要加 $
  • 变量名最好有意义，能够做到见名知意
  • 当变量名由多单词组成时，最好使用下划线分割或者使用驼峰命名法
  • PHP变量区分大小写

以下变量名哪些是正确的：

双引号和单引号的区别

双引号中的变量会被解释为值

单引号中的变量会被解释为字符串

单引号的执行速度会快于双引号的速度

常量

保存不会发生改变的数据(如：3.1415， 路径等)时，最好使用常量。

常量的使用方法：

1) 声明: define(常量名, 常量值, 大小写区分标志); 

true（不区分）/false（区分/默认）;

2) 调用: echo 常量名;
